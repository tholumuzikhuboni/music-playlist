## 🚀 About This Project

This is a **Music Playlist Web App** that allows users to interact with and explore a playlist of uDumakahle music tracks. The app features a dynamic playlist, a toggleable sidebar, and the ability to play and pause music. It is designed with vanilla HTML, CSS, and JavaScript.

## ✨ Key Features

-   🎶 **Dynamic Playlist**: Explore a curated list of uDumakahle tracks with all song details.
-   ▶️ **Playback Controls**: Simple and intuitive play/pause functionality.
-   📋 **Toggleable Sidebar**: Easily show or hide the full playlist to focus on the music.
-   📱 **Responsive UI**: A clean interface that adapts seamlessly to any screen size, from mobile to desktop.
-   ℹ️ **Track Information**: View the title, artist, and album of the currently playing song.

## 🚀 Getting Started

This project is built with vanilla web technologies and requires no special dependencies to run.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/tholumuzikhuboni/music-playlist.git](https://github.com/tholumuzikhuboni/music-playlist.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd music-playlist
    ```

3.  **Open `index.html` in your browser** to launch the application.
